Применение метода резольвенты к классической задаче трёх тел

Идея Ремизова реализуется численно: движение разбивается на множество малых шагов, каждый из которых решается как локальная линейная система. Ниже — симплектический «kick-drift-kick» шаг и его композиция во времени.

Δt 0.01
Шаги 0
Энергия 0
тело m₁ тело m₂ тело m₃

Локальная резольвента (один шаг)

На шаге Δt ускорения считаются постоянными, а движение приближается линейной системой. Симплектический вариант разбивает шаг на полуприкиды:

Kick v ← v + ½·a(r)·Δt
Drift r ← r + v·Δt
Kick v ← v + ½·a(r)·Δt

Композиция таких шагов при Δt → 0 реализует идею «бесконечной киноленты» Ремизова, а формула Ли–Троттера гарантирует сходимость.

Разбиение оператора

Движение разделено на оператор свободного хода (A) и оператор сил (B). Их чередование имитирует экспоненту e^{(A+B)Δt}.

  • A: ṙ = v, v̇ = 0
  • B: ṙ = 0, v̇ = F(r)

Контроль точности

Уменьшайте Δt, чтобы повышать точность. Визуально видно сохранение структуры орбит и энергию, характерную для симплектических методов.

Алгоритм композиции

  1. Вычислить ускорения от всех пар тел.
  2. Сделать полушаг для скоростей.
  3. Обновить координаты по новым скоростям.
  4. Пересчитать ускорения и завершить полушаг.
